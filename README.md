# 简介
就看天气——是一款遵循 **Material Design** 风格的只看天气的APP。
- 卡片展现（当前天气情况，未来几小时天气情况，生活建议，一周七天概况）
- 缓存数据，减少网络请求，保证离线查看
- 内置两套图标（设置里更改）
